## termux-service
### 配置服务
```shell script
pkg install termux-services

mkdir -p $PREFIX/var/service/robot/log
ln -sf $PREFIX/share/termux-services/svlogger $PREFIX/var/service/robot/log/run
vi $PREFIX/var/service/robot/run

###
#!/data/data/com.termux/files/usr/bin/sh
# 替换为你的 manage.py 所在目录（例如 $HOME/my_project）
cd $HOME/data/an_taskflow/

# 替换为你的实际命令（保持 python manage.py cmd --xx 不变，可改为 python3）
exec python manage.py runserver 0.0.0.0:8000
###

chmod +x $PREFIX/var/service/robot/run
chmod 755 $PREFIX/var/service/robot
chmod 755 $PREFIX/var/service/robot/log

# 退出当前会话重新进入, 临时启动
sv up robot

# 开机启动
sv-enable robotsv

```


### termux-service常用命令
sv-enable <服务名>	启用开机自启并立即启动服务  
sv-disable <服务名>	禁用开机自启并停止服务  
sv up <服务名>	临时启动服务（不设置自启）  
sv down <服务名>	停止服务（不改变自启状态）  
sv restart <服务名>	重启服务  
sv status <服务名>	查看服务状态
